# Text Summarization System

This project is a text summarization system built using FastAPI and the Hugging Face Transformers library. It allows users to generate concise summaries of long-form text in different styles, such as bullet points, paragraphs, or headlines. It also includes functionality to evaluate the quality of the generated summaries using ROUGE metrics.

## Features

- Accepts long-form text as input.
- Generates summaries in different styles:
  - Bullet Points
  - Paragraph
  - Headline
- Uses prompt engineering to enhance the quality and consistency of summaries.
- Evaluates summary quality using ROUGE metrics.
